# 🥕 Del Campo a la Mesa 🛒

Sistema CRUD desarrollado en PHP y MySQL para el manejo de productos agrícolas, pensado para vendedores rurales.

## ✨ Funcionalidades principales

- ✅ **Registrar productos** con nombre, descripción, precio e imagen.
- 📝 **Editar productos** ya registrados.
- ❌ **Eliminar productos** de forma segura.
- 🔐 **Login para vendedores** con acceso personalizado.
- 🖼️ **Visualización de productos** en tarjetas con imagen y detalles.
- 🎨 Interfaz con **Bootstrap 5** para una experiencia moderna y responsiva.

## 🗂️ Estructura del proyecto

login/

## ⚙️ Tecnologías usadas

- 💻 PHP 8
- 🛢️ MySQL / phpMyAdmin
- 🎨 Bootstrap 5
- 🖼️ HTML + CSS
- 🗂️ XAMPP para entorno local
